# Knowledge Ascension

## Project Overview
Knowledge Ascension is a multiplayer educational study game that lets users test their knowledge using interactive quizzes while they compete with others on a leaderboard.

## Tech Stack
Frontend: React + Vite
Backend: Node.js + Express
Database: PostgreSQL
Version Control: GitHub

## System Architecture

User → React Frontend → Express API → PostgreSQL Database

## Installation

1. Clone repository
2. Install dependencies

npm install

3. Start backend

npm run server

4. Start frontend

npm run dev

## Features

- User authentication
- Multiplayer quiz sessions
- Score tracking
- Leaderboard
